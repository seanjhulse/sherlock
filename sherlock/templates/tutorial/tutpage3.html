SherLOCK - Brief Introduction to Network Protocols


To begin deciphering the meaning behind the Nodemap, we must first begin by understanding what the individual components mean.

First, let's address the numbers that appear on the edges of the Nodemap.
These are port numbers. They describe what ports on a server are active entrances and exits for incoming/exiting data. Certain ports are assigned to perform different functions.
The most common port numbers you can expect to encounter are:

 - 80
 - 443

Port 80 is by default used for HTTP (Hyper Text Transfer Protocol) connections. Port 443 is used for HTTPS (Hyper Text Transfer Protocol Secure) connections.

Port 443 represents the transfer of data from the server to the local host (you) and the randomly assigned port number represents the return transfer of data from you to the server.
