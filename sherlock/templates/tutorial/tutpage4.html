SherLOCK - Context Menu


As interesting as it may be to watch your screen flash color to signify data packet transfer from server to client and back, it doesn't allow for much interaction from the user. To address this we added a context menu.

To access the context menu, you can either left click and hold a node until the menu appears or you can right-click a node to access the context menu.


Moving your cursor towards the ring allows you to select different functions offered by the Nodemap.

To the top left of the context menu,

copies the node's IP address to your clipboard.

To the top right of the context menu,

allows you to see other clients connected to that particular server node.

And finally to the bottom of the context menu,

allows you to scan for all other open ports on the server node.
